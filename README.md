# 🩺 Indian Liver Health Prediction using Machine Learning

## 📌 Overview
This project develops a Machine Learning model to predict liver disease risk among Indian patients using clinical and biochemical parameters to support early diagnosis and preventive care.

---

## 🚀 Project Highlight 

- Liver disease cases in India often go undiagnosed until advanced stages due to limited early risk screening tools.  
- Build a predictive model to classify patients as liver disease positive or negative using clinical attributes.  
- Performed data cleaning, handled missing values, conducted EDA, and trained Logistic Regression and Random Forest models with cross-validation and performance tuning.  
-  Achieved **87%+ accuracy**, identifying key biomarkers influencing liver disease risk and enabling early detection insights.
